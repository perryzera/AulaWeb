Comandos Git

Configuração

git init - Inicia um projeto git

git clone "url" - Clona um repositório Git

git remote add "remote" "url" - Adiciona uma "branch local"

Baixando Atualizações

git pull --allow-unrelated-histories "remote" "branch" - Permite pegar alterações de um projeto com o histórico diferente do projeto local (indicado para quando já existe um projeto no git e um projeto local)

git pull "remote" "branch" - Pega as ultimas alterações comitadas no servidor do Git

Fazendo Atualizações

git add . - Adiciona todas as mudanças ao stage

git commit -m "Texto sobre" - Realiza um commit das mudanças

git push "remote" "branch" - Envia as mudanças para um repositório git

Criando uma Nova Branch (Deve ser criada pelo GitHub)

git checkout -b "nova-branch" - Cria uma nova branch

git branch - Verifica em qual branch você está

git checkout "branch" - troca de branch

Extras

git remote rm "remote" - Remove um repositório remoto

git remote -v - Lista todos os repositórios remotos

Config

git config --global user.name - Ver usuario configurado

git config --global user.email - Ver email configurado

git config --global --unset user.name - Apaga o Usuário

git config --global --unset user.email - Apaga o email
